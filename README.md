# TYPE BRO Typing
Inspired by [Monkeytype](https://monkeytype.com)

A high-performance, minimalist, and aesthetic typing application built with React and Vite.

## Features

- **Premium Aesthetics**: Multiple curated themes (Neon, Sepia, Nord, Carbon).
- **Live Performance Graph**: Real-time WPM trend visualization.
- **Tactile Feedback**: Integrated mechanical keyboard sound effects.
- **High Precision**: Industry-standard WPM and Accuracy calculations.
- **Zen Mode**: Distraction-free typing experience.

## Credits

TYPEBRO is heavily **inspired by Monkeytype**. It aims to be a study in minimalist design, performance, and buttery-smooth user interaction.

## Getting Started

1. Clone the repository.
2. Run `npm install`.
3. Start the dev server with `npm run dev`.

## Deployment

This project is configured for one-click deployment to **Vercel**.
